Isomorphic JavaScript "The popularity of Node has lead to an interesting development known as
Isomorphic JavaScript. This involves having the same JavaScript code that can be
run either on the client or server side: if a browser is unable to run the code, it
can be run on the server and downloaded, or if the server is unavailable, the code
can be run on the client."
