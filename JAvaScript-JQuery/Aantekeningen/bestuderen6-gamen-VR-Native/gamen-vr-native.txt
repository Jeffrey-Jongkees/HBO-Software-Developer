GAMEN

- Flash was voorhen de meest gebruikte technologie voor het ontwikkelen van games op internet
- OmdatFlash veel mobiele apparaten niet ondersteund, heeft JS zijn plaats ingegnomen als de enige technologie
- JavaScript verkleint de grootte van de code die moet worden geschreven en zorgt voor betere spelprestaties
- In de loop van de tijd zijn veel JavaScript-bibliotheken en op JavaScript gebaseerde engines ontwikkeld die het maken van games mogelijk maken zoals:
    Phaser.js, 
    melon-js en visual-js.
    
Sommige JavaScript-frameworks ondersteunen 2D- of 3D-gaming, terwijl andere beide ondersteunen. 
Enkele populaire 3D-gaming frameworks zijn:
    Babylon.js
    Three.js 
    PlayCanvas.js.
